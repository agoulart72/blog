Installing Node.js / NPM

Node.Js is a JavaScript runtime built on Chrome's V8 JavaScript engine. npm is the world’s largest software registry.
January 19, 2020

Installing node in linux is pretty simple : 

 
$ sudo apt-get install nodejs
After installing node.js verify and check the installed version. You can find more details about current version on node.js official website. 
$ node -v 
v13.3.0
Also, check the npm version 
$ npm -v 
6.13.4
However, there's always a "catch". Versions from the ppa repository are not always up to date. Then, you can install a package called nvm. You can find it on its page on github : 
https://github.com/nvm-sh/nvm
There, as you may see, you can download and install nvm using the following command : 
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash
To download, compile, and install the latest release of node, do this: 
$ nvm install node # "node" is an alias for the latest version
